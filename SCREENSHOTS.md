   ```bash
   npm run dev
   ```

2. **Open browser**: Navigate to http://localhost:5173/

3. **Capture each view**:
   - Task 1: Click "Task 1: Company Comparison" tab
   - Task 2A: Click "Task 2A: Heatmap" tab

   - Or use browser screenshot feature
